- El Modelo ADDIE (Diseño Instruccional)Es el marco metodológico más utilizado en la creación de cursos en línea. Funciona de manera estrictamente lineal a través de cinco fases sucesivas
- Secuencia de Contenidos por MódulosEstructura clásica de un aula virtual donde el acceso al contenido está bloqueado hasta que se completa la etapa anterior.
